This is originally my sophomore curriculum design work.<br>
This is a forum website base on jsp.<br>
<br>
**Usage**:
- Git clone:<br>
  `$ git clone https://github.com/huiyifyj/Jsp-Forum.git`
- Deploy on Tomcat.
- Import Sql file.
- Open your browser:<br>
  `http://127.0.0.1:8080/forum`
